# 🌳 GenesisOS — Sovereign Asgard

GenesisOS este o distribuție NixOS personalizată, construită pe principii de suveranitate digitală, modularitate și performanță. 

## 🚀 Status Curent: Asgard Portabil
Sistemul rulează pe hardware real (Laptop) cu suport complet pentru **NVIDIA** și un kernel adaptiv care asigură stabilitatea modulelor externe.

## 🏗️ Arhitectură
- **Kernel:** Latest (ZFS/NVIDIA compatible fallback).
- **Interfață:** Hyprland (Wayland) cu estetică Tokyo Night.
- **Management:** Nix Flakes + Home Manager.
- **Workflow:** Keyboard-centric, Remote Deployment.

## 🛠️ Instalare Rapidă (Ignition)
Momentan sistemul este în faza de „Forjare Manuală”. Scriptul de `ignition.sh` este în curs de dezvoltare.
